## ğŸ¨ CUSTOMIZATION TIPS

### Change the Gold Color

Edit `styles.css`:
```css
:root {
  --accent-gold: #CD9935;  /* Change this! */
}

Try:
#D4AF37 - Metallic gold
#B8860B - Dark goldenrod  
#FFD700 - Bright gold
#C9A961 - Champagne gold
```

### Adjust Background

```css
:root {
  --bg-primary: #FAFAF9;  /* Cream */
}

Try:
#FFFFFF - Pure white
#F5F5F5 - Light gray
#FFF8F0 - Warm white
#F0F0F0 - Cool gray
```

### Change Font Pairing

```css
/* Luxury options: */
Playfair Display + Lato
Crimson Text + Work Sans
Libre Baskerville + Source Sans

/* Modern options: */
Poppins + Inter
Raleway + Open Sans
Nunito + Roboto
```
